# Design Rationale

Tone: premium, modern, trustworthy product app.

Palette:
- Dark base with emerald/cyan accents for freshness + utility focus.

Typography:
- Geist variable font for clean, product-grade readability.

Layout:
- Left app rail + main content panel to mirror future mobile IA modules.
- Card-based sections for easy component extraction into native/mobile later.

Micro-interactions:
- Subtle hover feedback on nav/buttons only; no noisy animation.
