# Offer Strategy

This is a premium ($1000-tier) webapp build direction, not a static brochure.

Positioning:
- Value is in personalized meal utility + coupon conversion + store-local relevance.

Conversion focus:
- Move users from onboarding/preferences to recommendation and redemption loops.
- Track retention with returning-user recommendation quality.

Handoff readiness:
- Feature routes and domain modules prepared for dev-team extension and mobile pivot.
