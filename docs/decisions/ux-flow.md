# UX Flow

Primary flow:
1. User lands on premium overview (value proposition + app entry).
2. User enters app and sets preferences/restrictions.
3. User discovers meals and saves relevant options.
4. Returning experience surfaces personalized recommendations.
5. User browses/collects coupons and generates QR for redemption.
6. User picks nearest store for localized availability.

Design intent:
- Keep app shell navigation persistent for low cognitive load.
- Keep route boundaries aligned with future mobile tabs/screens.
